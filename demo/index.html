JS form validation module

This module is supposed to provide support for validating almost any form. Based on the form, and validation rules that can be defined for each field, it should return all fields that pass the validation and if the form is valid or not.

It also includes printing messages below invalid fields and adding a class to invalid fields.

Basic form

Basic form with two text input fields

Form with select element

Form with one text field and one select field. It is useful to use required and in rules when validatin select fields, in order to restrict non-permitted values from being selected.

Email validation

If you need, you can validate a field to match email format, and print out nice validation message.
