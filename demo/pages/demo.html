Demo Forms

On this page you can explore how each of the validation rules work on real forms.

You can check live validation for each field, which will enable the field to get validated as you type.

"Required" rule

If you want to make sure a field gets filled.

"String" rule

If you want to make sure a field contains a value of type string.

"Minimum string length" rule

If you want to make sure a field has text with a minimum defined length.

"Maximum string length" rule

If you want to make sure a field has text with a maximum defined length.

"Email" rule

If you want to make sure a field has a valid email string.

"In" rule

If you want to limit which values the field can contain, you can use the "in" rule.

"Between" rule

If you want to make sure a number fields value is between some values.

"Greater than" rule

If you want to make sure a number fields value is greater than some value.

"Greater than or equal" rule

If you want to make sure a number fields value is freater than or equal to some value.

"Less than" rule

If you want to make sure a number fields value is less than some value.

"Less than or equal" rule

If you want to make sure a number fields value is less than or equal to some value.

"Date" rule

If you want to make sure a fields number is a valid date.
